# 05 Admin Observability and Support Alignment

## Goal

Ensure admin and support teams can understand why a user is being prompted to complete their profile.

## Why This Matters

If a user says:

- `I already filled everything`
- `why am I still seeing this block`

support needs visibility into the actual missing fields and gating logic.

## Recommended Admin Visibility

Admin should be able to see:

- profile completion percentage
- missing required fields
- whether the user is blocked from charts or transits
- whether onboarding is still incomplete

## Support Alignment

This should align with the existing support and admin systems so staff can:

- explain what is missing
- guide the user to the right page
- identify bad or stale completion state

## Deliverables

- admin visibility requirements
- support troubleshooting checklist
- alignment with existing completion and onboarding data
