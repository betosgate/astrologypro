# 06 SEO Routing and Indexation Strategy for Dual Landings

## Goal

Define how the two landing models should coexist without cannibalizing each other or creating duplicate search posture.

## Required Concern

If both of these exist:

- service-only landing
- service plus specific diviner landing

then canonical routing and indexation rules must be explicit.

## Recommended Rule

- service-only pages should target broad service intent
- diviner-specific pages should target branded plus service intent

The SEO model must decide:

- which page is canonical for broad service queries
- which page is canonical for diviner-branded queries

## Deliverables

- route hierarchy recommendation
- canonical and sitemap rules
- duplicate-content prevention strategy
