# Task: User Details View Modal

## Objective
Add a "View Details" modal to the Admin User List that displays all relevant information for a selected user without navigating away from the list.

## Requirements
- [ ] **Trigger**: Add a "View" or "Eye" icon/button for each user in the list.
- [ ] **Modal Presentation**: Use a clean, well-structured modal (Dialog/Sheet) to present the user's data.
- [ ] **Data to Display**:
    - Full Name and Profile Picture (if available)
    - Email, Phone, and User ID
    - Role and Detailed Status (Active/Inactive, Membership Type, etc.)
    - Timestamps: Joined At, Last Login
    - Audit Trail: Recent Login History and Admin Notes (summary)
    - Role-specific metadata (e.g., certification status for diviners, referral codes for advocates).
- [ ] **Interactions**: Include quick links to the Edit Page or Delete action from within the modal.
