## Goal

Define backend/API contract used by Single signup birth-location autocomplete.

## API Contract

- URL: `https://astrologypro.com/api/admin/astro/city-search`
- Method: `POST`
- Request:
```json
{ "q": "kolk" }
```
- Response fields needed:
1. `results[].label`
2. `results[].lat`
3. `results[].lng`
4. `results[].timezone.offset_string`

## Backend Rules

1. Response shape must stay stable for frontend mapping.
2. Empty results should return an empty `results` array.
3. Invalid request should return safe validation error.

## Acceptance

1. API returns label/lat/lng/timezone fields required by frontend.
2. Contract supports signup validation flow.
