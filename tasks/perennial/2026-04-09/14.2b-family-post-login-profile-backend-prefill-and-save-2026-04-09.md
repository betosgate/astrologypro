## Goal

Define backend prefill and save behavior for Family post-login completion.

## Save Rules

1. Save by authenticated `user_id` context only.
2. Do not trust payload `user_id`.
3. Support GET prefill + PATCH save.
4. Support incremental partial saves.
5. Return full family profile object and completion state.
6. Keep member-to-record mapping stable during update.
