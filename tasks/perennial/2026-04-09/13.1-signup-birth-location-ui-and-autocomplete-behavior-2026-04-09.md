## Goal

Design post-login completion UI behavior for `birth_location` field with autocomplete.

## UI Rules

1. Add visible field label: `Birth location`.
2. User typing triggers debounced search.
3. Show suggestion list from API.
4. On selection, store mapped location object.
5. Clear mapped lat/lng/tzone if user edits text after selecting.

## Accessibility

1. Field supports keyboard navigation.
2. Suggestions are selectable by keyboard and mouse.

## Acceptance

1. Suggestions appear for valid query.
2. Selected location is visibly confirmed.
