## Goal

Integrate city autocomplete UI for birth place selection in Single signup.

## UI Behavior

1. User types city query in signup form.
2. Debounced API call.
3. Dropdown suggestions from `results`.
4. On selection, map to:
   - `birth_location_label = label`
   - `birth_lat = lat`
   - `birth_lng = lng`
   - `birth_tzone = timezone.offset_string`

## Validation

1. Require a selected city object, not plain free text.
2. Reject submit if `birth_location_label` missing.

## Error States

1. API failed: show retry message.
2. No results: show `No matching cities`.

## Acceptance

1. City search returns options and selection sets birth location fields.
2. Signup cannot complete with only typed city text and no selected result.
