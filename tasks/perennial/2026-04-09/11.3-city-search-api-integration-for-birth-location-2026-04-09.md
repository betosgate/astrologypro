## Goal

Integrate city autocomplete for birth place selection in post-login completion form.

## API Contract

- URL: `https://astrologypro.com/api/admin/astro/city-search`
- Method: `POST`
- Request:
```json
{ "q": "kolk" }
```
- Response fields needed:
1. `results[].label`
2. `results[].lat`
3. `results[].lng`
4. `results[].timezone.offset_string`

## UI Behavior

1. User types city query.
2. Debounced API call.
3. Dropdown suggestions from `results`.
4. On selection, map to:
   - `birth_location_label = label`
   - `birth_lat = lat`
   - `birth_lng = lng`
   - `birth_tzone = timezone.offset_string`

## Validation

1. Require a selected city object, not plain free text.
2. Reject submit if `lat/lng/tzone` missing.

## Error States

1. API failed: show retry message.
2. No results: show "No matching cities".

## Acceptance

1. City search returns options and selection sets all birth location fields.
2. Form cannot complete with only typed city text and no selected result.
