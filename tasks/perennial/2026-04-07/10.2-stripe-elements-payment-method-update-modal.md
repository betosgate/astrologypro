# Task 10.2: Stripe Elements Payment Method Update Modal - 2026-04-07

- Status: Planned
- Priority: P2
- Scope: `@stripe/react-stripe-js` Integration, New SetupIntent API
- Task File: `tasks/perennial/2026-04-07/10.2-stripe-elements-payment-method-update-modal.md`

## Objective

Allow users to update or add a new payment method directly on the dashboard without leaving the screen, using a custom Stripe Elements component wrapper mimicking the legacy AngularJS UI.

## Implementation Steps

1.  **API Setup Intent Endpoint (`/api/community/billing/setup-intent`)**:
    - Build a POST endpoint returning a Stripe `client_secret` from `stripe.setupIntents.create({ customer: customerId })`.
    - Accept payload: `{ customer_id: string, email: string }`.
    - This replaces the old `cart_management/subscription-payment` API from the legacy project.

2.  **Stripe Elements UI (`<UpdatePaymentModal />` & `<MembershipCard />`)**:
    - Build a "Payment" Update popup triggered directly from the `<MembershipCard />` via a new "Update Payment Method" button.
    - Install and configure `@stripe/react-stripe-js` and `@stripe/stripe-js`.
    - Build a dark-themed modal displaying the Stripe `<PaymentElement />` (Card, Apple Pay, Ideal, etc.).
    - Include the orange `Pay & Subscribe` (or "Update Payment Method") action button at the bottom.

3.  **Success State Screen (`<PaymentSuccessOverlay />`)**:
    - Upon successful Stripe confirmation (`stripe.confirmSetup`), grab the `setupIntent` ID.
    - Dismiss the payment modal and show the exact success overlay (Gold mandala background with green checkmark).
    - Display the Setup ID (`seti_xxxx`), Name, Email, and exact localized Date.
    - Include the orange `Go to Dashboard` button to close the overlay and refresh the UI state.

## Done Definition

- The Stripe JS library is successfully integrated into the Next.js Community layouts.
- Users can input secure payment elements directly in the popup.
- The UI mimics the provided screenshot references exactly.
- The backend API securely attaches the newly collected Setup Intent to the existing Stripe Customer profile.
