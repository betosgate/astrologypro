## Goal

Define and implement the structural layout of the dashboard with a dedicated left-side sidebar for navigation.

## Required Behavior

1. The portal should have a fixed or collapsing left-side sidebar.
2. The main content area should be intelligently padded or shifted to make room for the sidebar.
3. The layout should be responsive, likely hiding the sidebar on mobile and using a different navigation method (e.g., bottom-nav or slide-out menu).
4. Use standard modern layout tools like CSS Grid or Flexbox.

## Tasks

1. Define the sidebar container in the layout.
2. Ensure the layout handles differing viewport widths gracefully.
3. Set the base structure for navigation links within the sidebar.

## Acceptance Criteria

- Sidebar exists on the left side.
- Sidebar does not overlap main content.
- Responsive states are defined for mobile.
