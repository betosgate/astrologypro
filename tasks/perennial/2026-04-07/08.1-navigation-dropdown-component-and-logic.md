## Objective

Build the underlying logic to support a multi-level (dropdown or accordion) navigation structure in the Perennial left-side sidebar. This will be used primarily for grouping the "Product" menu items.

## Implementation Steps

1.  **Sidebar Menu Structure**:
    - Update the `Sidebar` or `Nav` component to handle an `items` array that can include `children`.
    - Handle the `isOpen` state for collapsible menus.

2.  **Collapsible Component**:
    - Create a small `CollapsibleNav` or `SubMenu` component.
    - Animate the opening/closing for smoothness.
    - Use a chevron icon (`v`) that rotates when the menu is open.

3.  **Active Route Detection**:
    - Ensure that if a child route (e.g., `/admin/perennial/product-categories`) is active, the parent "Product" menu is also highlighted/expanded.

## Done Definition

- The logic exists to define nested navigation items.
- Parent items can trigger a collapsible sub-menu.
- Chevron rotation works according to state.
- Route-aware logic correctly expands menus based on current URL.
