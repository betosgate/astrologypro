## Objective

Replace the text-only instructions view in the ritual detail page with an interactive auto-playing video component. This requires modifying `src/app/community/rituals/[id]/page.tsx` to handle immersive media playback.

## Implementation Steps

1.  **Video Player Component (`<RitualVideoPlayer />`)**:
    - Design a clean, immersive media player container.
    - If `currentInvocation.video_url` exists, render the `<video>` element. 
    - Fallback elegantly to static text instructions if no video exists for a specific minor step.

2.  **Autoplay & Lifecycle Logic**:
    - Tap into standard HTML5 video events (e.g., `onEnded`).
    - When a video finishes naturally, automatically progress the user to the next ritual step via the existing `handleNext()` / `patchStep()` logic.

3.  **UI Polish**:
    - Keep the textual instructions below or overlaying the video for accessibility and reference.
    - Overlay controls and the current step's name.

## Done Definition
