# Task 05 - Remove Family Members Label And Use Icon Tooltip

- Status: Planned
- Priority: P1
- Area: Perennial / Community Dashboard / Top Summary UX
- Page Route: `/community`

---

## Objective

Reduce horizontal noise by removing the visible `Family Members` text label and relying on the people icon as the row anchor.

## UX Direction

- Remove the visible label text
- Keep the people icon
- Add a tooltip on hover/focus:
  - `Family members`

## Why

The chips themselves already communicate the row content.
The text label takes space and increases density without adding enough value.

## Acceptance Criteria

- [ ] `Family Members` text is removed
- [ ] The icon remains visible
- [ ] Tooltip provides accessible clarity
