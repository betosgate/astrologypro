## Database Audit SQL

Check `community_members`:

```sql
SELECT column_name, data_type, character_maximum_length, is_nullable
FROM information_schema.columns
WHERE table_schema = 'public'
  AND table_name = 'community_members'
  AND column_name IN (
    'id', 'email', 'full_name', 'first_name', 'last_name', 'phone', 'gender',
    'date_of_birth', 'birth_time', 'birth_city', 'birth_country', 'state',
    'city', 'zip', 'address', 'relationship_status', 'relation_type',
    'intake_data', 'occupation', 'membership_status', 'membership_type',
    'plan_type', 'onboarding_completed'
  )
ORDER BY column_name;
```

Check `community_family_members`:

```sql
SELECT column_name, data_type, character_maximum_length, is_nullable
FROM information_schema.columns
WHERE table_schema = 'public'
  AND table_name = 'community_family_members'
  AND column_name IN (
    'id', 'full_name', 'date_of_birth', 'birth_time', 'birth_city',
    'birth_country', 'relationship', 'notes', 'invite_email', 'user_id'
  )
ORDER BY column_name;
```
